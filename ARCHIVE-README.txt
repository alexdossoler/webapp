# Project Intake Form - Enhanced Version

Complete Next.js TypeScript intake form with admin dashboard and file uploads.

## Quick Setup:
1. npm install
2. npx prisma db push
3. npx prisma db seed  
4. npm run dev

## Access:
- Form: http://localhost:3000
- Admin: http://localhost:3000/admin/login
- Credentials: admin@example.com / admin123

## New Features:
- File attachment support in admin dashboard
- Secure self-hosted file uploads
- Image preview capabilities
- Enhanced lead management
